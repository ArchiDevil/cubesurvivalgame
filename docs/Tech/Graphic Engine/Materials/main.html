diffuseColor: color(R, G, B)

Use this property to set diffuse color for material. Diffuse color is used while lighting calculation.

specularColor: color(R, G, B)

specularIntensity: {real}

Use this property to set specular color for material. Specular color is used while lighting calculation.

Specular intensity is used to define angle of specular component. Less is darker specular component.

reflective: {boolean}

If reflective property is true, material will reflect environment. Using this property can decrease FPS on low-end video systems.

transparent: {boolean}

transparency: {real}

Use this property to define material's transparency properties. If transparent set to true, you can use transparency property.

diffuseMap: {texture}

diffuseMap: array({textures})

Diffuse map is a property to control you diffuse colors for every pixel of object. You can also use array of textures if you use DX10+/OpenGL3.3+ API and three texture coordinates in you vertex.

normalMap: {texture}

normalMap: array({textures})

Diffuse map is a property to control you diffuse colors for every pixel of object. You can also use array of textures if you use DX10+/OpenGL3.3+ API and three texture coordinates in you vertex.

luminosityColor: color(R, G, B)

Use this property to set luminosity color for material. Useful for light sources. Material doesn't affect any lights around.

specularMap: array({textures})
